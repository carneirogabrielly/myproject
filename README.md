# Cata Triangulo

Jogo 2D top-down onde você precisa coletar 4 triângulos em menos de 20 segundos enquanto desvia de inimigos.

**Projeto da disciplina Jogos Digitais 2026.1 — Insper**

## Como jogar

- **Mover**: WASD ou setas do teclado (controle Xbox / joystick também funciona)
- **Objetivo**: coletar os 4 triângulos antes do tempo acabar
- **Evite os inimigos** — eles perseguem ou patrulham e tiram sua vida no contato
- Você tem **3 vidas**

## Mecânicas

- **Tempo**: 20 segundos para terminar a partida. Se acabar, é Game Over.
- **Vida**: 3 HP. Ao perder tudo, é Game Over.
- **Inimigos**: uns patrulham em padrão fixo, outros perseguem o player.
